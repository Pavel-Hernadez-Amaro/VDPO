## Summary of changes in this version

This is a minor update (0.2.0). It adds:

* Support for partially observed functional data (`ffpo()`, `ffpo_2d()`,
  `po_fit()`, `po_2d_fit()` and the corresponding data generators).
* A multivariate functional principal component analysis for variable domain
  data (`mfpca_vd()`) with a `plot()` method.
* Two new vignettes.
